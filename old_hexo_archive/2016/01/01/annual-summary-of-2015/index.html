      2015 Summary

      Post on 2016-01-01

      很久没写文章，博客都长草了。

      时间过得好快，大四已经过了半个学期了。没人跟我提这事，我也没什么自觉，只有闲暇时偶尔才会发现大学生涯只剩几个月了，只能默默感叹时光飞逝。

      这个博客建成也已经有两年多了，虽然没什么内容，也不好看，没折腾太多，我还是通过这个博客得到了不少东西，比如友情，比如知识，还有各种建议。最近也是苦口婆心劝后辈们去搭个自己的小博客。（这段只是一点小感慨，绝无废博之意，我还会时不时写点东西发上来的）
      记得开博之初，我说开这个博客的目的是「发表自己的言论与观点」和「记录自己的生活、学习与工作的点滴」。现在看过来博客大体是后者类型的，可能因为我比较没有头脑吧。
      所以，这次也是一篇「记录自己的生活、学习与工作的点滴」的流水帐。用行话来说就是 tl;dr。

      去年也写了类似的总结，不过不同的是，去年每个月都有自我总结下，所以干了什么很清晰明了，而今年变动比较大，所以到了五月过后例行的总结就坑了。所以这次还要先对五月过后的事情做下梳理。

      迟到的 6~12 月总结

      这一段时间开始了我人生中的第一份工作，其实就是苦逼加班码农。从 PHP 玩到 Java，然而两边都学得很肤浅，只到勉强用来写业务的程度，还不到深刻理解的境界。

      工作的事情很枯燥的，讲太多就像发牢骚。
      期间有事请假回家、回学校几趟，也不是很值得一提的事情。

      不过这些时间并不无趣，自己租了房子后，我有了自己完全独立支配的时间和空间，爬山、逛西湖、看书、画画、宅、开脑洞、睡觉，想干啥就干啥。也交了不少公司的朋友，融入进团体中，也是件很开心的事情。
      并没有什么十分不满意的事情。

      这几个月的总结就是这样，下面是正式的年度流水帐。

      书及年度最佳书籍

      真心感觉出学校后就没怎么好好读过书，感觉比去年读的少多了，不过整理了下发现还是看了几本。2015 年读过的书 - 豆列

      1. 黄金时代 （上个寒假拜读了下王小波的书。修行不够没读出什么来）
      2. 松本行弘的程序世界 （讲 Ruby 的好书，不过还是修行不足，虽然看过但现在已经忘光了）
      3. 程序员的思维修炼 （我只记得挺有意思的）
      4. JavaScript高级程序设计（第3版） （这本书的效果：能 看懂 JS）
      5. 游戏开发中的人工智能 （学校课程相关，看得挺认真的）
      6. 黑客与画家 （不错的故事书）
      7. 情商：为什么情商比智商更重要 （情商有稍微一点的提高吧，大概）
      8. 代码整洁之道 （好书，主 Java）
      9. Python编程实战 （实践性的书，看了个大概）
      10. 大型网站技术架构 （对我这菜鸟来说一些概念总结得不错，不过被同事黑了）
      11. 三体 系列三本 （难得的国产好书，脑洞够大）

      双十一花了 ￥600+ 来买书，买了好多书，到现在为止没看完几本 Orz，嘛，算是明年半年份的书吧。

      个人感觉今年看的书整体没去年的让我印象深刻，或许是我没有认真的去研读吧。去年那批算是很强烈的改变我的三观。
      对我来说，今年的年度最佳书籍是——「三体」！准确来说是「三体」系列的三本书。「三体」脑洞之大，让我大开眼界，简直是国产科幻的希望。
      在这我就不啰嗦安利「三体」，看过的都知道其强大之处。我快成了大刘的粉丝了，已经买了大刘的另外几本书，会慢慢给看完的~

      电影及年度最佳电影

      今年算是迄今为止看看得最多的一年了，寒假在家看了 n 部电影，工作后公司每个月组织一次包场看大片。
      总的算起来，今年应该看了 40+ 的电影吧，这里就不一一列出来了。

      然后说说年度最佳电影吧，那就是在家看的「肖申克的救赎」。公司看的都是当时的热门大片，大片做为消费品一般没什么内涵，我看完并没什么深刻的感触。「肖申克的救赎」则不然，不知道比现在的商业片好到哪去了，具体观影感受看 以前写的观后感。
      （要不是「幽灵公主」我看过很多遍了，不想把老货搬出来，我每年都会选她的！她永远是我的第一！）

      动画及年度动画 top3

      我今年看了几部动画？我一时也说不清，每个季度都追了不少新番，不过有近半的都被我在前面几集弃了。总的来说应该也有不少了，毕竟这是我屈指可数的几个消遣之一呀。
      感觉如今的番越来越不好看了，只有一几部比较突出的。（可能是我口味变重了）

      年度动画 top3：

      1. 寄生兽 剧情、音乐都特赞！即使漫画看过来看动画也不会觉得腻。从深度上来说不会输给近年的大部分的作品，是难得一见的好作品。
      2. 迷糊餐厅第三季 看了几年终于迎来大结局。おめでとう、おめでとう。
      3. 终物语 现在看的是情怀，依旧还是很喜欢。小说后面出的就没时间再看了。

      漫画及年度漫画 top3

      对动画的要求变高让我空出了不少时间转战漫画，漫画可以方便的在手机上离线看，因此漫画看得会比动画多点。今年看过各种热门冷门、清新重口的漫画不计其数，不过智商不高的我没法很好品味漫画真正的艺术、内涵，所以会倾向脑洞、搞笑型的漫画。

      年度漫画 top3：

      1. 战国妖狐 水上悟志老师的作品剧情很赞！人物刻画入木三分！他的其它作品如魂环、惑星公主蜥蜴骑士，都是不错的作品。他的作品就是那种看完意犹未尽的那种。
      2. 超能力者齐木楠雄的灾难 神奇的设定+搞笑，我只感在家看，因为会笑出声来，永远不够看的那种类型。
      3. GrandBlue 个人认为是年度最搞笑的漫画，原作是井上坚二老师，也就是《笨蛋，测验，召唤兽》的作者。剧情什么的，说出来就没意思了。

      技能

      今年技能见长了？并没有点亮多少技能。
      因为开始实习，自由学习的时间变少了，所以今年新点的技能主要是工作需要的 PHP 和 Java 技能，还有一点点的服务器相关的技能，少得可怜。而且这些技能并没多么深入的学习，遇到问题只能各种求助大神，弱爆了 Orz。

      新的一年要想办法多点好玩或者实用的技能了，然后把工作用技能学得深入一点。

      成就

      1. 工作。我也没想到我这么弱，居然那么快就找到了实习工作。人生成就 get。
      2. Github 100 Day Streak 达成，其实有 147 天的 Streak 的，不过后面工作了就没继续了 Orz。
      3. 画画 50 张达成。（兴趣使然，然而拿不出手）

      剁手

      开始赚钱了，有钱就任性了点，剁手买了不少东西。

      • 双十一买了好多书 - 放书架吃灰
      • AKG K450 和 K420 - 平民耳机，还不错
      • 鼠标 霜冻之蓝 - 不错的鼠标，当时是想买来玩 OSU! 的来着，不过手残换鼠标也无济于事…
      • 数位板 Intuos 影拓 CTL680 - 入门级的数位板用来入门
      • LG Nexus 5X - 用了三年半被刷了几百次的 i9300 寿终正寝，买了 Google 儿子情怀机。

      朋友

      今年遇到不少贵人相助，线上线下又交到了不少朋友，我真是个幸运的家伙。
      只能说，明年还请多指教了。

      不想夸下海口，只求 2016 年能变得更强。
      我也会勤更新 github 和博客了，确实荒废太久了。
