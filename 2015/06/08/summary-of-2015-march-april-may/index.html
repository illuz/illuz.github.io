      summary-of-2015-march-april-may

      Post on 2015-06-08

      三四月份生活变动很大，从学校跑杭州实习了，事情不少，本来打算两个月一块写了，没想到拖了这么久，结果现在只能三个月一块写了（成季度总结了，叹气）。

      然而我并不打算长篇大论，这只是闲扯，我也不会写像之前那样列清单出来一个个划掉。

      工作

      面试的事情可以看我之前写的面经，不过之后会把那篇文章删了，因为这样赤裸裸地署名公开人家辛辛苦苦想出来的面试题，不仅对出题方不是很尊重，而且听说会掉人品。
      反正我发过，有兴趣去 github 找之前的版本看吧。

      在这公司实习还不错，被分配到综合业务部门，刚遇到大活动，加班稍微有点多，不过过后就没那么厉害了。
      在这主要做的是居然是 PHP！我也没万万没想到啊。虽然被灌输过「PHP 是最好的语言」思想，我还是会干一行爱一行的，我会好好领略 PHP 的美妙的。而且它只是一个大系统的一小部分而已。
      够我玩一阵了。

      杭州

      在杭州住了一个月了。

      这阵子空气完全不行，就如你我心中想的那样。

      步行去西湖玩了两趟，风景还可以，有空可以多跑跑。

      剁手

      我是比较会自制的类型，几次想剁手买东西，都犹豫半天打消了念头。（所以手机用了三年，不过也快淘汰了）
      在出租房时还是下手买了网件 WNDR4300，不过最近这边网络不是很稳定，回头再刷个 OpenWrt 玩玩。

      大的剁手还是等我有钱了再好好考虑了。

      书

      从学校出来好像还没安下心来看书过，需要自省。
      在学校倒是看了不少的样子，现在要回忆，我只能说『程序员的思维修炼』这本书讲得还不错，其它我已经忘了。

      番

      这几个月看过不少的动漫、漫画，不知从何说起。
      感觉快过了看新番的年纪了都。
      （是时候找经典老番复习复习了）

      游戏

      这几周一直在玩「崩坏学园2」这个游戏，不得不说是个不错的手游，平衡性做得不错，还有钉宫的声优让人无法自拨，我也是玩疯了。
      玩起来入迷太误事了，所以我玩到 60 多级（还没爆到神器）就怒删游戏了。

      OS

      这几个月先后入坑了 Archlinux 和 OSX。

      Linux 我只接触过 Ubuntu 和 Kubuntu，很久前就想转 Archlinux 了，于是先在虚拟机里熟悉了一段时间，然后找了个时间干掉 Windows 换上了 Archlinux，但我的双显卡也是花了不少时间折腾。
      总之 Archlinux 可定制性很高，用起来还是比较满意的，至于 Windows 就丢 VirtualBox 里了。

      如果不是公司提供 Mac 机，我想我可能要过几年才用得上 Mac。
      刚开始玩 OSX 还是不是很习惯的，键盘和触控板都用得很不习惯，想不到现在很喜欢它的键位和触控板。不得不说比其它电脑人性化多了（不吐槽新 MacBook 的键盘）。不过最近用大拇指按 command 键按得有点疼，可能是我姿势不对。
      不仅键盘和触控板，Mac 整机体验起来都很不错，以后有钱了可以考虑入个自己用的。

      总之俩系统我都很喜欢。
      工作都用的 Mac，而且 Mac 续航能力很强，可以撑一晚上，所以大部分时间用的 Mac。

      Github

      我的「编程马拉松」跑了 166 天后，决定不再硬撑下去了，因为工作没时间玩，而且每天为了 push 而 push 太无趣了。
      虽然知道可以写个脚本每天定时跑下来 push，甚至可以用一些工具一次把所有日子点绿，不过这太没意思了。
      我还是慢慢来吧。

      v2ex 记录

      我停止了「每周v2ex记录」这个系列。
      详细可以看之前的文章：http://illuz.github.io/2015/05/31/v2ex-150531/
      可能我要多思考，写些有深度点的文章了。

      (End)
