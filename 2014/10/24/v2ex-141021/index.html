      每周v2ex记录[14-10-20~14-10-24]

      Post on 2014-10-24

      由于周末有事外出，不能上网，所以这周只到周五了。
      虽然天数很少，但这周意外的有料:)。

      1. 自学了 Android 基础，该怎么进阶？

      等俺Android基础也自学完了，也来进阶进阶><。

      • 摘录1：
        Android学习之路

      • 摘录2：

        1. 挖一个复杂项目的坑开始填.
        2. 开始看一些开源项目
        3. 开始看Android的系统源码
          建议先定好方向，然后从某个方向上开始进一步深化。

      • 摘录3:

        做一个完整的、复杂一点的 App。从 UI 到数据库，会遇到各方面的问题，然后深入学习各个部分。最后，写一份几万字的总结

      2. Android 快速入门看什么？？

      先入门再谈进阶吧哈哈。
      虽然下面资料我都有了，但是还是记录下吧，如果有新回复再补充。

      • 官方 tutorial
      • Android学习之路
      • 《Learning Android 2nd Edition》

      3. 发一个零基础学 Python

      Gitbook地址： http://looly.gitbooks.io/python-basic/
      Github地址： https://github.com/looly/python-basic/
      看评论好像不错的样子，可以推荐给需要的基友看。

      4. 恐怖游轮评分 8.2，很想看，但是胆子小，会吓尿吗？

      由一个恐怖电影钓出了很多重口的东西哈哈。
      虽然我不重口，不过难得推荐的那么多，我也记录下，好像都是关于循环类的。

      无线循环类无线循环类的，可以从不那么恐怖的《Edge of Tomorrow》、短片《ROOM 8》、短片《Mouse-X》等入手。
      近几年来的恐怖片佳作当属2012的《The Cabin in the Woods》。
      AVG电子小说类游戏《Ever17》
      电影《The Shining》
      欢乐版本的《土拨鼠日》汤姆汉克斯
      觉得不够刺激可以看《致命弯道》系列

      5. Code Marathon(代码马拉松)，坚持才能看到彩虹

      很酷的东西，这种装逼的东西我喜欢><。
      待我比赛打完了就开始搞吧。（又是一个神坑的节奏）

      6. 在线思维导图

      虽然我有安装了MindManager和XMind了。（听说XMind最新版能更新Evernote了，不错嘛，有机会在Windows也装下）
      但是在线的脑图还是很酷的。
      几个不错的在线思维导图get：

      • https://coggle.it/ 虽然是国外的，但现在不用翻墙
      • http://naotu.baidu.com/ 百度居然有这东西…
      • https://kozea.github.io/umlaut/ 用json作图
      • http://www.memofon.com/ 用markdown写，神奇的想法

      7. 10 分钟以内的励志视频（微电影）

      虽然自己不是很感冒，但是还是有可能拿去给别人看的。
      所以记录下。

      8. 黑科技 for 小学生

      https://photomath.net/
      黑科技get。妈妈再也不用担心我的学习了。（虽然我用不上了）
      给钱还能一步一步解，不过收费就只能表示呵呵了。

      9. Android 上展示 pdf

      虽然里面的帖子都歪楼了，不过之后说不定有大神回复，先收藏说不定能用上。
      让我想起以前收藏的一个“开源的，可以直接浏览 pdf 的网站程序”帖子。

      收藏好几篇关于编码的很棒的文章：
